  ## Logistic Regression
  - Regression (회귀) : 입력에 대한 출력값이 연속적인 실수값으로 출력  
  - y =wx + b 인 선형 식을 Activation 함수에 대입해 Non-Linear하게 만듬.  
  - **수치가 아닌 확률로 출력**하고 싶을 때 사용. Linear Regression 사용 시 확률이 1이 넘어가는 현상 발생.
